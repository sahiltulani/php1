Functions

  • Function Definition: A function is a reusable block of code that performs a specific task. It is defined using the function keyword followed by the function name and parentheses.
  • Creating and Invoking User-Defined Functions: User-defined functions are created by the programmer. They are invoked (called) by using the function name followed by parentheses. Example: 
    function myFunction() {
    // code to be executed
}
myFunction(); // calling the function
  • Formal Parameters vs. Actual Parameters:
    • Formal Parameters: Variables listed in the function definition that accept values.
    • Actual Parameters: The values passed to the function when it is called.
  • Function and Variable Scope: Scope determines the visibility of variables:
    • Local Scope: Variables defined within a function are accessible only inside that function.
    • Global Scope: Variables defined outside any function are accessible anywhere.
    • Static Variables: Retain their value between function calls.
  • Recursion: A function that calls itself to solve a problem, typically with a base case to terminate the recursion. Example: 
    function factorial($n) {
    return ($n === 0) ? 1 : $n * factorial($n - 1);
}
  • Library Functions: Built-in functions provided by PHP that can be used to perform common tasks, such as string manipulation, mathematical calculations, and array handling.

String Handling

  • Interpolation with Curly Braces: PHP allows variable interpolation within double-quoted strings using curly braces for complex variables. Example: 
    $name = "John";
echo "Hello, {$name}!"; // Outputs: Hello, John!
  • Characters and String Indexes: Strings can be indexed similarly to arrays. For example, $str[0] accesses the first character.
  • String Operators: The concatenation operator (.) is used to join strings. Example: 
    $greeting = "Hello, " . "world!";

  • String Functions: Built-in functions for string manipulation (e.g., strlen(), str_replace(), strpos(), etc.).
  • Formatting Strings: Functions like sprintf() and printf() are used to format strings according to specified format strings.
  • Comparing and Searching Strings and Substrings: Functions like strcmp(), strpos(), and strstr() are used for comparing and searching within strings.

Arrays

  • PHP Arrays: A data structure that holds multiple values. PHP supports indexed (numerically) and associative (key-value) arrays.
  • Creating Arrays: Arrays can be created using the array() function or the shorthand [] syntax. Example: 
    $arr = array(1, 2, 3);
$assoc = ["key1" => "value1", "key2" => "value2"];
  • Accessing Array Elements: Accessed using square brackets with the index or key. Example: 
    echo $arr[0]; // Outputs: 1
echo $assoc['key1']; // Outputs: value1
  • Multidimensional Arrays: Arrays containing other arrays. Example: 
    $multiArr = [[1, 2], [3, 4]];
  • Inspecting Arrays: Functions like print_r() and var_dump() help inspect the contents of an array.
  • Deleting from Arrays: Use functions like unset() to remove elements. Example: 
    unset($arr[1]); // Removes the second element
  • Iterating with each() and foreach():
    • each(): Returns the current key and value pair and advances the array pointer. Note: Deprecated as of PHP 7.2.
    • foreach(): Used to loop through each element of an array. Example: 
      foreach ($arr as $value) {
    echo $value;
}
  • Iterative Functions:
    • current(): Returns the current element.
    • next(): Advances the internal pointer to the next element.
    • prev(): Moves the pointer to the previous element.
    • reset(): Moves the pointer to the first element.
    • end(): Moves the pointer to the last element.
